Welcome to cantkeepup app

Goal of the app

The target of the app is to make the user issue commands from his browser address bar. Commands like 's keywords' which will google for keywords, 'shorten url' which returns a shortened url, or 'ocvs keywords' which will search in the codebase of OpenCV on github. Currently there is a set of global commands that can be used by everyone, and in order to create your own commands you have to register first and then from the dashboard you can add your commands.

How to install

Chrome

Goto settings then under the search section press Manage search engines now we will add cantkeepup here, scroll down until you find 3 empty boxes. In the first box Add a new search engine type cantkeepup. In the second box Keyword type cantkeepup. In the last box URL with %s in place of query type http://cantkeepup.herokuapp.com/?q=%s. Then press enter, and make it the default one by pressing the Make default button that appears when you hover over it.
